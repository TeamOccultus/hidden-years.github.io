# 进阶材料
## 铁
**铁**是一种常见且坚固的材料，可以合成工具、武器、盔甲。

| 属性 | 数值 |
| :--- | :--- |
| 近战伤害 | 7 |
| 耐久 | 250 |
| 耐久倍率 | 15 |
| 挖掘速度倍率 | 6 |


## 秘银
![纹理](/items/silver_ingot.png)

**秘银**是在钻石之前最珍贵与坚韧的材料，可以在地下挖掘秘银矿石并熔炼获得。

秘银锭可以合成工具、武器、盔甲。

| 属性 | 数值 |
| :--- | :--- |
| 近战伤害 | 7 |
| 耐久 | 800 |
| 耐久倍率 | 25 |
| 附魔能力 | 50 |
| 挖掘速度倍率 | 7 |

## 钻石
**钻石**是一种广为人知的、坚硬的蓝色宝石，极为稀有。

钻石可以合成工具、武器、盔甲。


| 属性 | 数值 |
| :--- | :--- |
| 近战伤害 | 8 |
| 耐久 | 1561 |
| 耐久倍率 | 33 |
| 挖掘速度倍率 | 8 |

## 魔晶
![纹理](/items/crystal_ingot.png)

**魔晶**是一种附魔能力极高的材料，通过熔炼附魔紫水晶碎片即可获得。

> [!TIP]
> 采集紫水晶块、紫水晶母岩会有概率掉落附魔紫水晶碎片。

魔晶可以合成工具、武器、盔甲。

| 属性 | 数值 |
| :--- | :--- |
| 近战伤害 | 8 |
| 耐久 | 1600 |
| 耐久倍率 | 35 |
| 附魔能力 | 60 |
| 挖掘速度倍率 | 8 |
