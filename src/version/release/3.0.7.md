## 新内容
- 加入了文海探源录任务章节，用于引导玩家获得阅读物
- 优化了百科的相关内容
  - 向百科中添加了盔甲赐福相关内容 
  - 完善了材料教程的内容 
  - 加入了 JSON UI 形式的游戏百科（内容暂时不完善）
- 加入了三种 BOSS 人偶
- 调整了职业相关机制
  - 行旅者满级后不再会赠送抉择晶石，需要击败红宝石之王以获得抉择晶石 
  - 向行旅者角色详细中添加了有关抉择晶石的相关提示 
  - 提高了职业升级所需的统一货币值 
  - 将高阶职业的升级物品改为魔法尘埃/魔法源质 
- 加入了圣辉之触
  - 牧师的专属武器
  - 由海晶砂粒和木棍合成而来
- 加入了骑着僵尸马的月蚀僵尸 
  - **注：在 26.10 以下的版本中，该特性不会正常运作**

## 修改
- 将盔甲赐福·不屈的触发概率由 100% 下调至 65%
- 优化了法老残影的 AI 意向 
- 允许一些方块含水 
- 修改了教程中有关于转职的相关文本 
- 下调了月蚀怪物的生成权重 
- 更新了阅读物的纹理 
- 更新了沙蚀苦力怕的模型、纹理和动画 
- 更新了试炼怪物的模型与纹理 
- 现在沙蚀苦力怕被闪电击中会直接爆炸，而非转换为闪电沙蚀苦力怕

## 技术性

- 加入了 hiddenyears:job_recovery 组件
