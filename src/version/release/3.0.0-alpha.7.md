本次更新利用 Moonlike Loom 大幅减小了文件体积。

<!-- more -->

## 修改
- 启用了新的打包方式
  - 音乐与包图标文件被压缩以减小体积
  - 数据驱动文件均被压缩
  - 效果：该测试版体积仅为上个测试版体积的 38%
- 降低了各个结构的生成频率 
- 整理了创造模式物品栏分组
- 更新了职业相关物品的纹理
- 更新了一些饰品的纹理
- 更新了殷红之花
- 更新了空饰品槽位的纹理
- 更新了雨神之冠的纹理
- 更新了原初护腿和靴子的物品纹理
- 更新了寒钢盔甲的物品纹理
- 更新了恐惧之石的纹理

## 修复
- 修复了所有金属方块创造模式物品栏位置错误的漏洞
- 修复了部分矿石没有被正确归类的漏洞
- 修复了武器匠交易表失效的问题 #53
- 修复了货币无法合成的漏洞
- 修复了红宝石头盔护甲值过高的漏洞
- 修复了狡兔之靴饰品种类错误的漏洞
- 修复了下界合金与钻石战锤中文名称错误的漏洞 #59
- 修复了每次进存档都多生成一本引导书的漏洞 #54
- 修复了魔晶盔甲纹理错误的漏洞
- 修复了巴别塔之碎片·赤沙没有名称的漏洞
