本次更新加入了魔法相关特性。

<!-- more -->

> 从该版本开始，我们将会逐步对隐藏之年内的游戏内容进行彻底的重构
> 
> 请注意，重构后的隐藏之年与旧版本隐藏之年**不兼容**

## 新内容
### 魔法
- 加入了创造模式物品栏魔法分类 
- 加入了魔法源质 
- 加入了传送石碑、传送钟表
- 加入了归乡晶石 
- 加入了归途宝石 
- 加入了老法杖 
- 加入了复合药水和魔法酿造机制 
- 加入了炼金机制
- 加入了老法师 
- 加入了老法师之家 

### 主菜单
- 加入了时匿怀表（隐年主菜单）
- 将职业菜单入口添加至主菜单中 
- 加入了统一货币值兑换货币的功能 

## 修改
- 增强了压缩矿石的战利品表丰厚程度
- 提高了 BOSS 血量 
- 精简了部分物品的名称 
- 降低了宝石商人小屋的生成概率 
- 将宝石商人小屋设置为仅平原可生成以规避生成错误 

## 技术性
- 加入了`hiddenyears:profile`组件 
- 修复了`ArmorTypr`组件类型错误的漏洞 
- 拆分了一些内建数据文件 
- 用Moonlike Loom替换Starock Cli 

## 修复
- 修复了饰品相关恶性漏洞  #60
- 修复了检测物品不会检查副手的漏洞 #61
- 修复了沙漠怪物攻击没有干旱效果的漏洞 
- 修复挖掘红宝石矿物之后`destroy_ruby_count`次数永远不会增加的漏洞
- 修复了法杖没有粒子效果的漏洞 
- 修复了牧者/农夫草帽使用时会抛出错误的漏洞
- 修复了金撬棍没有名称的漏洞 
- 修复了金制勇者徽章没有名称的漏洞 
- 修复了可以通过章节奖励无限刷取货币的漏洞 
- 修复了受侵之兽任务无法完成的漏洞 
- 修复了试炼怪物会自相残杀的漏洞 
- 修复了金石就砺任务无法打开的漏洞
