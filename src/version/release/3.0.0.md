# 隐藏之年 「天边的迦万涅尔」主题更新
## 游戏内容
### 材料体系

- 重新设计了材料体系
  - 加入了闪耀之石，挖掘荧石概率掉落
  - 加入了恐惧之石、恐惧之魂
  - 加入了古战残骸、远古碎片
  - 加入了魔晶材料、秘银材料、精炼岩材料
  - 现在原初合金（最高等级）具有了抗火功能
- 为所有类型材料添加了对应盔甲
- 允许所有盔甲进行纹饰操作
- 加入了碎石机
- 移除了仿制材料

### 工具与武器

- 重新设计了工具和武器
  - 移除了小刀、锥、锤
  - 加入了重锤、战锤、锯
  - 加入了各种强化弓、强化箭
  - 加入了各种魔法弩、魔法箭
- 加入了魔法锻造台  
- 加入了弓匠台
- 现在使用锯、重锤等工具挖掘对应方块时有几率获得额外增产
- 修改了下界合金工具与武器的获取方式，使其与原版趋同

### 饰品

- 加入了五种饰品槽和对应的饰品
- 加入了古遗的试炼所和古遗的试炼石
- 加入了契约晶石和契约粉末
- 加入了试炼怪物

### BOSS

- 提高了所有 BOSS 的血量
- BOSS 现在不会在和平模式消失
- 各个BOSS不再可以被命名牌命名且不受窒息、闪电伤害
- 优化了创世者之怒的技能效果

### 职业
- 加入了四个初阶职业以及抉择晶石
- 加入了所有高阶职业
- 加入了回溯宝石
- 加入了所有职业的技能

### 魔法
- 加入了魔法的尘埃
- 加入了魔法源质
- 加入了传送石碑、传送钟表
- 加入了归乡晶石、归途晶石 
- 加入了老法杖 

**老法师**
- 加入了老法师 
- 加入了老法师之家 

**炼金台**
- 炼金台现在会发出等级为 8 的光；
- 将徒手挖掘炼金台所需时间延长为 12 秒；

**复合酿造**
- 加入了复合药水
- 加入了魔法酿造机制 

**沙漠酿造**
- 加入了沙漠药壶
- 允许玩家使用沙漠特产（如沙蚀之骨）和基础沙漠药壶酿制不同类型的沙漠药壶

**炼金**
- 加入了炼金机制

### 遗迹
- 加入了殷红教堂，作为红宝石之王的 BOSS 战场地
- 加入了红宝石试炼密室和其中的雷光秘宝
- 加入了受囚之牢，精灵会在其中生成
- 加入了宝石商店，宝石商人会在其中自然生成
- 加入了沙漠棺材
- 加入了回生祭所 
- 加入了故国遗迹 
- 加入了尼罗特兵站 
- 加入了沙漠补给站 
- 适当调整了沙漠结构的生成间距
- 允许拼图结构含水
  - 尽管如此，结构在浸水时仍会出现一些意料之外的行为（如不渲染水面）

### 月蚀怪物
- 加入了月蚀僵尸
- 加入了月蚀骷髅
- 加入了月蚀旅者及其四个变种
  - 上述月蚀怪物的攻击力均会随月相变化，月亮越接近满月，攻击力越高
- 加入了圣光之剑，可对月蚀怪物造成额外伤害
- 为痛苦之剑添加了对月蚀怪物造成额外伤害的功能
- 为痛苦之剑、圣光之剑添加了附着物
- 更新了痛苦之剑的物品纹理

### 盔甲效果
- 从代码中启用了不屈、复生、雨之神的祝福三种盔甲效果
- 为带有盔甲效果的物品添加了提示

### 石胚与精炼岩锭
- 加入了石胚，可用于获取精炼岩锭
- 修改了精炼岩锭的获取方式

### 生物群系

- 加入了殷红平原
- 加入了樱花平原

### 教程、任务与背景故事
**任务**
- 重构了任务系统
- 优化了序章第一幕、第二幕任务
- 加入了一批新任务
  - 用于引导玩家探索月蚀怪物、复合酿造等 3.0.0 新特性
- 为一些旧任务重写了任务描述

**故地异国行纪**
- 加入了「故地异国行纪」机制
- 开局自动获取，内容会随任务推进而持续更新
- 旨在引导玩家了解背景故事

**教程**
- 加入了游戏百科 
  - 可以在隐藏之年主菜单中的「图鉴」一栏中找到
  - 目前该特性正在测试中，可能会包含一些小问题

**角色故事**
- 加入了玩家角色故事
  - 会随着玩家

### 主菜单
- 加入了时匿怀表（隐年主菜单）
  - 将职业、任务和教程入口添加至主菜单中 
  - 加入了统一货币值兑换货币的功能 

### 交易
- 优化了所有商人的经济交易
- 拓宽了钻石币在交易中的使用场景 

### 命令

- 添加了命令`/hiddenyears:credit`，用于显示《隐藏之年²：天边的迦万涅尔》鸣谢名单
- 添加了命令`/hiddenyears:migrate`，用于将玩家任务迁移到新版本中
- 添加了命令`/hiddenyears:profile`，用于打开玩家主菜单

### 其他

- 调整煤炭粉末的燃烧时间
- 重新加入了欢迎消息
- 加入了三个新唱片
- 降低了各个结构的生成频率 
- 添加了任务迁移功能
  - 当玩家有任务可以迁移时，游戏会向其发送提示消息
  - 可使用`/migrate task`命令进行迁移
- 降低了压缩矿石的生成数量和尝试次数
- 调低了压缩矿石的生成高度
- 将压缩矿石的挖掘等级提高至钻石
- 将囚灵的生成亮度修改为`[0,7]`

## 资源更新

- 更新了项目的图标与 Logo
- 加入了「创世者」的愤怒的 BOSS 战曲目与生物音效
- 「平原遐想」曲目现在会在创造和生存模式下播放
- 重新绘制了大部分纹理
- 优化了大部分生物的纹理、模型与动画

## 技术性更新
### 破坏性更改

- 移除了 2.X 版本所有可用的物品标签；
- 将附加包命名空间由`hy`改为`hiddenyears`
- 使用 Occultus SDK 构建脚本内容
- 将沙蚀苹果的 ID 改为`hiddenyears:sandsandcaust_apple`
- 将沙蚀蜘蛛眼的 ID 改为`hiddenyears:sandsandcaust_spider_eye`
- 弃用了历战宝袋、任务书

### 数据驱动

- 使用原版的`minecraft:geometry.full_block`构建完整方块
- 将配方、物品、方块文件的格式版本更新到 1.21.130 版本
- 改进了挖掘与物品等级系统，现在使用纯原版数驱实现
  - 因此，挖掘赐福和诅咒机制也被移除
- 使用拼图结构构建所有遗迹
- 加入了`hiddenyears:is_artifact`物品标签
- 向金属粒添加了`minecraft:matel_nuggets`标签
- 加入了`trial_monster`族

### 合成标签

- 开放了`alchemy_table`合成标签，适用于炼金台；
- 开放了`bowyers_bench`合成标签，适用于弓匠台。

### 本地化

- 使用 JSON 文件储存本地化字符串；
- 标准化了所有本地化字符串的格式，以方便与高亮显示相关附加包兼容；
  - 需要注意的是，高亮显示相关附加包大多对原版 UI 进行了修改，因此一些可能的兼容性问题我们仍然无法修复。
- 加入了英语、俄语、日语、繁体中文、文言文的翻译
  - 这些翻译还不完整，欢迎到 https://translate.codeberg.org/projects/hidden-years/ 贡献翻译
  
### 打包
- 启用了新的打包方式
  - 音乐与包图标文件被压缩以减小体积
  - 数据驱动文件均被压缩

### 物品组件

### 物品组件
**hiddenyears:hidden_effect_food**
- 添加自定义模拟效果
- 支持破伤风、失血、脱水、干旱效果
- 可配置持续时间和效果等级

**hiddenyears:effective_food**
- 添加状态效果
- 支持效果分组（`ALL`/`GOOD`/`BAD`）
- 可配置粒子效果和持续时间

**hiddenyears:custom_tool**
- 自定义工具类型
- 支持镐、斧、剑、铲、锄等类型
- 可设置为虚拟工具（无耐久消耗）

**hiddenyears:custom_weapon**
- 自定义武器系统
- 支持虚拟武器模式
- 无耐久消耗特性

**hiddenyears:tool_type**
- 设置工具类型
- 支持锤子、撬棍、锯子等特殊工具
- 影响方块交互行为

**hiddenyears:weapon_type**
- 设置武器类型
- 支持剑、锤子、匕首等类型
- 影响战斗行为

**hiddenyears:custom_crossbow**
- 自定义弩系统
- 支持拉力等级配置
- 可自定义弹药列表

**hiddenyears:arrow_present**
- 箭矢预设系统
- 支持日光、雷光、烈焰、寒钢等类型
- 特殊效果配置

**hiddenyears:trophy_bundle**
- 历战宝袋系统
- 支持硬编码和脚本战利品表
- 可配置战利品来源

**hiddenyears:exp_food**
- 食物经验值系统
- 可自定义经验值数量

**hiddenyears:coin**
- 货币系统
- 支持统一货币值
- 可配置货币价值

**hiddenyears:staff**
- 范围法杖系统
- 可配置伤害和范围
- 支持粒子和音效效果

**hiddenyears:job**
- 职业系统
- 支持行旅者职业
- 可选择保留或移除旧职业

**hiddenyears:return_gem**
- 传送道具系统
- 支持多种绑定方式
- 可配置传送音效

**hiddenyears:specific_damage**
- 特定伤害系统
- 可对不同生物族群设置额外伤害
- 支持伤害范围配置

**hiddenyears:structure_placer**
- 结构放置系统
- 支持高度限制
- 可配置放置动画和偏移

**内部组件**
- `hiddenyears:blessing_of_isis`：雨神祝福
- `hiddenyears:complex_potion`：复合药水
- `hiddenyears:adventurer_note`：故地异国行纪
- `hiddenyears:profile`：时匿怀表
- `hiddenyears:job_skill`：职业技能

### 方块组件

**hiddenyears:interactable**
- 标记方块为可交互方块

**hiddenyears:ore_type**
- 设置矿石类型（ruby/normal）
- 影响挖掘行为

**hiddenyears:vault**
- 配置宝库功能
- 支持战利品表、转换方块、钥匙和音效

**hiddenyears:boss_spawner**
- 设置BOSS祭坛
- 支持BOSS ID、转换方块、钥匙
- 可配置客户端事件和淡入淡出效果

**hiddenyears:trial_stone**
- 设置试炼之石
- 支持三种状态：等待激活/激活/熄灭
- 可配置战利品表、转换条件、钥匙和音效
