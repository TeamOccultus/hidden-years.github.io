本次更新加入了大量沙漠结构，修复了多个漏洞。

<!-- more -->

## 关于 Alpha 测试阶段的说明

这将是 3.0.0 最后一个 Alpha 测试版，3.0.0 所要包含的大部分内容已经得到了实现。

接下来的 Beta 阶段将主要围绕漏洞修复、性能优化和引导完善来开展。

## 新内容
### 沙漠结构
- 移植了沙漠商店 
- 移植了沙漠探索者营地 
- 移植了幽匿实验室 
- 移植了沙蚀蜘蛛的乐园 
- 移植了地下行宫的一角 
- 移植了往世之梦
- 移植了沙漠地牢 
- 移植了无名坟墓
- 加入了沙漠棺材
- 加入了回生祭所 
- 加入了故国遗迹 
- 加入了尼罗特兵站 
- 加入了沙漠补给站 
- 适当调整了沙漠结构的生成间距
- 允许拼图结构含水
  - 尽管如此，结构在浸水时仍会出现一些意料之外的行为（如不渲染水面）

### 职业
- 修改了一部分职业的实现 
- 实现了唤灵法师的技能并加入其对应召唤物
- 现在职业菜单会显示职业的固有天赋和主动技能的介绍 
- 为所有职业的升级条件启用了统一货币值
- 为行旅之石添加了使用音效 

### 其他
- 加入了寒钢矿石
- 减小了月蚀怪物的攻击对象范围 
- 将月蚀怪物与月蚀旅者的生成逻辑进行了合并 
- 修改了月蚀怪物的伤害机制
- 更新了制作人员名单

## 修改
- 优化了流沙怪物的纹理
- 优化了红宝石之王的模型、纹理和动画
- 优化了法老残影的模型、纹理和动画
- 优化了木乃伊的纹理、模型和动画 
- 优化了红宝石怪物的索敌机制 
- 更新了尼罗特行商的纹理 

## 技术性
- 公开了`hiddenyears:structure_placer`组件 
- 使用 OSDK构建 BOSS独白 
- 向统一货币值系统的内部处理过程中添加了取整机制

- 修复了沙蚀蜘蛛眼 ID 拼写错误的问题
- 修复了沙蚀苹果 ID 拼写错误的问题 
- 修复了部分技能无法正确被触发的问题 
- 修复了流沙守卫没有音效的问题
- 修复了月蚀怪物模型错误的问题 
- 修复了任务界面无法正确处理带描述的物品的问题 
- 修复错误骨套纹理 
- 修复了农夫草帽部分情况下缺失翻译的问题 
- 修复了玩家无法将进阶职业升级到 11~15 级的问题
