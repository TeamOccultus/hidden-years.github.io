## 修改
- 优化了宝石商人、隐藏商人、图书管理员和沙漠游商的经济交易 
- 优化了盔甲匠、牧师、制箭师和武器匠的经济交易 
- 优化了弓和弩的创造模式物品栏排列 
- 优化了原初盔甲的客户端资源 
- 为原初盔甲添加了抗火功能 
- 优化了创世者之怒的技能效果与音效
- 各个 BOSS 不再可以被命名牌命名且不受窒息、闪电伤害 
- BOSS现在不会在和平模式消失 
- 为切割机加入了粒子效果 

## 技术性
- 更新了所有方块和生物群系的格式版本到`1.21.130`

- 开放了`hiddenyears:staff`组件
- 开放了`hiddenyears:hidden_effect_food`组件 
- 向各BOSS添加了`boss`族
